# BidORBoo



* To get started 
  * make sure to have NodeJs Npm and yarn installed globally 
  * make sure to get the configuration folder from one of the developers and extract it in cd bidorboo/server
  * git clone this repo
  * cd bidorboo/client
  ``` 
  yarn run install
  yarn run start
  ```
  * cd bidorboo/server
  ```
  yarn run install
  yarn run dev
  ```
